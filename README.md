# TVDE Practice Test App

## Problem
Preparing for the Portuguese TVDE exam required repeatedly answering hundreds of questions from a PDF.

## Solution
I converted the questions into JSON and built an interactive quiz application that simulates the exam.

## Features

✅ Multiple-choice questions
✅ Randomized quizzes
✅ Score calculation
✅ Instant feedback
✅ Simple interface

## Technologies

- Python
- Streamlit
- JSON
